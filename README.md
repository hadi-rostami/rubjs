# RubJS

> یک فریمورک مدرن، زیبا و کاملاً غیرهمزمان (Asynchronous) برای تعامل با API رسمی روبیکا با استفاده از زبان جاوااسکریپت

---

### معرفی

**RubJS** فریمورکی پیشرفته و در عین حال ساده برای کار با API روبیکا است که به شما امکان می‌دهد به‌راحتی از طریق یک حساب کاربری (کلاینت سفارشی) با روبیکا ارتباط برقرار کنید. این فریمورک به صورت کامل غیرهمزمان طراحی شده تا بهترین کارایی را در برنامه‌های مدرن جاوااسکریپتی ارائه دهد.

---

### نمونه کد اولیه

```javascript
const { Client, Utils, Filters } = require("rubjs");

const bot = new Client("bot");

const object_guid = "object_guid";

const customFilter = (msg) => msg.object_guid === object_guid;

bot.onMessageUpdates([customFilter, Filters.isText], async (message) => {
  await message.reply(`Hello ${Utils.Italic("from")} ${Utils.Bold("RubJS")}!`);
});

bot.run();
```

## ویژگی‌ها
> **آماده و سریع**:
به سادگی با نصب از npm آماده استفاده است و در کمترین زمان می‌توانید شروع به توسعه کنید.

> **سادگی در عین قدرت**:
رابط کاربری ساده و قابل فهم که در عین حال تمام قابلیت‌های پیشرفته و رسمی API روبیکا را پوشش می‌دهد.

> **طراحی زیبا و مدرن**:
انتزاع مناسب از جزئیات سطح پایین، برای تمرکز روی منطق برنامه و توسعه سریع‌تر.

> **کاملاً غیرهمزمان**:
تمامی عملیات به صورت async/await انجام می‌شود تا تجربه توسعه روان و بهینه داشته باشید.

> **دسترسی کامل به API**:
امکان اجرای تمامی عملیات رسمی کلاینت روبیکا و استفاده از امکانات پیشرفته در برنامه شما.
